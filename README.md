## Run Locally

<br>

Clone the project

```bash
  git clone https://github.com/itsdevdeepak/my-med-identity
```

Go to the project directory

```bash
  cd my-med-identity
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
